# 우리 자산 노트

배우자와 공유할 수 있는 개인 자산 기록용 React + TypeScript 웹앱입니다. 예금, 청약, 금, 실버바, 국내주식, 해외주식을 사용자가 직접 입력하고 원화 자산과 달러 자산을 별도로 합산해 보여줍니다.

## 실행 방법

```bash
npm install
npm run start
```

Vite 개발 서버가 실행되면 브라우저에서 표시되는 로컬 주소를 열면 됩니다. 기본 주소는 `http://127.0.0.1:5173`입니다.

프로덕션 빌드는 다음 명령으로 확인할 수 있습니다.

```bash
npm run build
```

## 주요 기능

- 원화 현금성 자산 등록: 자산명, 금융기관명, 금액, 메모
- 귀금속 등록: 보유 중량 g, 매수 평균단가 KRW/g, 현재 단가 KRW/g
- 국내주식 등록: 종목명, 종목코드, 보유 수량, 평균단가 KRW, 현재가 KRW
- 해외주식 등록: 종목명, 티커, 보유 수량, 평균단가 USD, 현재가 USD
- 원화 자산과 달러 자산 별도 합산
- 매수원금, 평가금액, 평가손익, 수익률 계산
- 환율 입력 시 달러 자산의 원화 환산 참고 금액 표시
- 카드형 자산 목록, 유형별 필터, 최근 등록순/평가금액순 정렬
- 자산 추가, 수정, 삭제, 상세 보기
- 비밀번호 입력 후 변경 잠금 해제
- 빈 상태 화면과 삭제 확인 Alert

## 데이터 저장 방식

서버 없이 브라우저 `localStorage`에 `assets` 배열을 JSON으로 저장합니다.

앱 최초 실행 시 샘플 자산 데이터가 한 번 저장됩니다. 샘플 데이터는 UI 확인을 위한 임의값이며 실제 투자 정보나 현재 시세가 아닙니다. 사용자가 샘플 데이터를 삭제하면 다시 자동 생성되지 않습니다.

## 변경 잠금

조회와 공유는 누구나 할 수 있지만, 앱 이름 변경과 자산 추가/수정/삭제는 비밀번호 입력 후에만 가능합니다. 현재 비밀번호는 `9207`입니다. 잠금 해제 상태는 브라우저 세션 동안 유지되며, `다시 잠그기` 버튼으로 즉시 잠글 수 있습니다.

이 방식은 브라우저에서 동작하는 간단한 변경 방지 장치입니다. 실제 계정 기반 보안이 필요한 경우 향후 서버 인증 또는 생체 인증 기능을 추가해야 합니다.

## 카카오톡 공유 방식

카카오 SDK를 직접 연동하지 않습니다. 대시보드의 `카카오톡으로 공유` 버튼을 누르면 브라우저의 Web Share API를 우선 사용해 OS 공유창을 열고, 사용자가 카카오톡을 선택해 텍스트 요약을 공유할 수 있습니다. Web Share API를 지원하지 않는 브라우저에서는 공유 메시지를 클립보드에 복사합니다.

공유 메시지는 실제 저장된 자산 데이터를 기준으로 생성하며, 자산이 없는 섹션은 생략합니다. 원화는 한국식 콤마와 `원`, 달러는 `$`와 소수점 2자리로 표시합니다.

## 수동 시세 입력

자동 시세 연동은 현재 MVP 범위에서 제외되어 있습니다. 모든 금액, 보유 수량, 현재가, 평균단가는 사용자가 직접 입력해야 합니다.

## 향후 개선 사항

- 국내주식 시세 API 연동
- 해외주식 시세 API 연동
- 금/은 시세 API 연동
- 배우자 계정과 실시간 동기화
- 비밀번호 또는 생체 인증 잠금
- 백업/복원 기능
